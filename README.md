# Jokenpo vs AI

Um jogo clássico de Pedra, Papel e Tesoura, mas com um diferencial: uma "IA" que tenta prever seu próximo movimento.

## Como funciona

O jogo utiliza um algoritmo de reconhecimento de padrões (ex: Cadeias de Markov simplificadas) para analisar o histórico de jogadas do usuário e tentar "counterar" a próxima escolha. Quanto mais você joga, mais difícil fica de ganhar.

## Funcionalidades

- Placar em tempo real.
- Reset de memória da IA.
- Animações de vitória/derrota.

## Stack

- HTML5 / CSS3
- JavaScript (Lógica de Jogo)

## Como Jogar

Abra o arquivo `jokenpo.html`.
